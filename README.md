# FormApp - Ứng dụng nhập liệu Google Sheet

Ứng dụng Android cá nhân giúp nhập liệu nhanh vào Google Sheet.

## Tính năng
- 📝 Tự cấu hình các trường dữ liệu tương ứng cột Google Sheet
- 📤 Gửi dữ liệu lên Google Sheet qua Apps Script API
- 💾 Lưu cấu hình trường dữ liệu vào bộ nhớ máy
- 🎨 Giao diện Material 3 chuyên nghiệp

## Build
APK được tự động build bởi GitHub Actions khi push code.
